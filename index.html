CURSO DE PROGRAMACIÓN WEB 1


A continuación, se presenta el formato de Guion para el desarrollo de la actividad de la Fase de Planeación y Análisis del curso Programación Web 1, desarrollando una aplicación usando herramientas como HTML5, CSS3, MySQL y PHP.


OBJETIVO GENERAL:


* Adquirir y aplicar conocimientos sobre sistemas gestores de base de datos con la ayuda de las nuevas tecnologías.


OBJETIVOS ESPECIFICOS:


* Generar conciencia sobre la importancia de los sistemas gestores de base de datos.

* Impulsar la utilización de los recursos tecnológicos disponibles para la enseñanza en base de datos.

* Promover el desarrollo de abstracción de información y solución a problemas de acuerdo a un caso dado, que permita definir el modelo de una base de datos.

*Solucionar problemas de administración de información que permitan acceder y realizar procesos CRUD en la base de datos (Crear, leer, actualizar y borrar).
